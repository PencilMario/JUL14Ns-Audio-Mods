# 日志栏功能测试指南

## 前置准备

1. **编译项目**
   ```powershell
   .\build.ps1
   ```

2. **安装插件**
   - 将生成的 `.ts3_plugin` 文件导入 TeamSpeak 3
   - 或将 DLL 手动复制到 TS3 plugins 目录

3. **启动 TeamSpeak 3**
   - 连接到至少一个服务器
   - 打开插件设置

## 测试步骤

### 测试 1: 查看初始化日志
1. 打开插件设置
2. 切换到"日志"标签页
3. **验证**: 应该看到：
   ```
   [HH:MM:SS.zzz] 插件初始化
   [HH:MM:SS.zzz] Device: <设备名称>
   ID: <设备ID>
   Type: Render (Playback)
   ```

### 测试 2: 显示当前设备
1. 在日志标签页，点击"显示当前设备"按钮
2. **验证**: 日志中应出现新的设备信息条目
3. **多次点击**: 更改系统默认设备后再点击
4. **验证**: 应显示新的设备名称

### 测试 3: 切换到默认设备
1. 在日志标签页，点击"切换到默认设备"按钮
2. **验证**: 日志中应出现：
   ```
   [HH:MM:SS.zzz] 用户触发：切换到默认设备
   [HH:MM:SS.zzz] 正在切换到：<设备名称>
   [HH:MM:SS.zzz] ✓ 切换成功
   ```
3. **检查 TS3 设置**: 确认播放设备已更改

### 测试 4: 清空日志
1. 日志栏中应有多条日志
2. 点击"清空日志"按钮
3. **验证**: 所有日志消失，日志栏为空

### 测试 5: 设备变化检测
1. 在 Windows 声音设置中更改默认播放设备
2. 点击"显示当前设备"按钮
3. **验证**: 显示新的设备信息
4. **关键验证**: 比较两次查询的设备名称应该不同

## 预期行为总结

| 操作 | 预期结果 |
|------|---------|
| 打开日志标签页 | 显示插件初始化信息和默认设备 |
| 点击"显示当前设备" | 显示当前系统默认设备的详细信息 |
| 点击"切换到默认设备" | 将 TS3 切换到系统默认设备并显示成功 |
| 更改系统设备后点击按钮 | 显示更新后的设备信息 |
| 点击"清空日志" | 清除所有日志文本 |
| 设备切换失败 | 显示错误信息"✗ 切换失败" |
| 无活跃连接时切换 | 显示"无活跃连接" |

## 故障检查清单

### 日志栏不显示初始化信息
- [ ] 检查 device_switcher.h 是否正确编译
- [ ] 检查 config.cpp 中的 appendLog 是否被调用
- [ ] 查看编译错误信息

### "显示当前设备"按钮无反应
- [ ] 确认按钮被正确连接到信号槽
- [ ] 检查 AudioDeviceListener 是否正确初始化
- [ ] 验证 WASAPI 头文件是否正确包含

### 切换设备失败
- [ ] 确认至少连接一个 TS3 服务器
- [ ] 检查日志中的具体错误信息
- [ ] 验证系统默认设备是否可用

### 编译错误
- [ ] 确保 Qt5 正确安装
- [ ] 检查所有必要的头文件是否包含
- [ ] 验证 WASAPI 相关的库是否链接

## 高级测试

### 压力测试
1. 快速点击"显示当前设备"多次
2. **预期**: 无崩溃，日志不断更新

### 设备热插拔测试
1. 连接新的音频设备（USB 耳机、蓝牙设备等）
2. 点击"显示当前设备"
3. **预期**: 显示新设备（如果系统已将其设为默认）

### 多连接测试
1. 连接多个 TeamSpeak 服务器
2. 点击"切换到默认设备"
3. **预期**: 所有连接的播放设备都被切换

## 性能验证

### 检查项
- [ ] 点击按钮无明显延迟（<100ms）
- [ ] 日志滚动流畅
- [ ] 没有内存泄漏
- [ ] 没有 CPU 使用率异常

### 性能检查方法
1. 打开任务管理器
2. 监控 TeamSpeak 进程的内存和 CPU 使用率
3. 多次点击按钮，确保使用率不会持续增长

## 日志格式验证

每条日志应遵循以下格式：
```
[HH:MM:SS.zzz] 消息内容
```

示例：
```
[14:23:45.123] 插件初始化
[14:23:45.124] Device: Realtek High Definition Audio
ID: {0.0.1.00000000}.{XXXX-XXXX-XXXX-XXXX}
Type: Render (Playback)
[14:23:50.456] 显示当前设备
[14:23:50.457] Device: Realtek High Definition Audio
...
```
